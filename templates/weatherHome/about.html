About Page

This website is made by Levi Herrera and it tells users the weather of locations they have opted in to receive information from.

You can see all the code and stuff from my projects github repo.

Why I made this site

The reason I created this website was because every morning I had to Google the weather in Aurora then the weather in Chicago and doing two Google searches every morning was too much. So, I was determined to build a website that would solve this problem. Out of that determination this website was born and now I can get the weather of whatever places I want so long as I have the necessary information.
